Detailed Description

Simple GUI region (not renderable).

Note
It is the simplest derivate of uiobject, so if you just need a widget that has a position and a size (like frame's title region), then this is the best choice.

◆ is_loaded()

bool lxgui::gui::uiobject::is_loaded ( ) const
inherited

Checks if this widget has been fully loaded.

Note
A widget that is not fully loaded still has all its core attributes set, hence can be considered as "fully constructed" from a C++ point of view. However, semantically, the object may need further steps to be complete, as designed by the UI designer. Therefore, form the UI's point of view, a widget is considered "complete" only if is_loaded() returns 'true' (see notifu_loaded()). Only then can the widget, e.g., react to or generate events.
